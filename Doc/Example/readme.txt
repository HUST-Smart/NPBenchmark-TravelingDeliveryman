

输入：
V:途中点的集合，表示外卖送往的地点的几何，集合中每个元素vi即一个地点也为一个集合（d, t, value)，它有三个元素，d表示地点编号(0表示起点），t表示先订外卖送达的时刻(此处为相对于出发点的相对时间），value表示外卖在限定时刻送达时的收益
S:表示边的集合，表示每两个地点之间的耗时的集合，集合中每个元素si也为一个集合(vi,vj,dij,tij)

注：若某地点的外卖限定到达时刻为t0，外面在时刻t送达(t>=t0),则相应的收益为 value = value - 0.5*（t - t0)

输出：
	总的收益value 以及路径的集合Path


示例图中的输入及输出：
输入：
V：(0, 0, 0）, （1, 15, 20), (2, 20, 25), (3, 10, 15)
S：(0, 1, 15, 10), (0, 2, ,17, 15), (0, 3, 6, 5), (1, 2, 13, 10), (2, 3, 18, 15)



legal输出：
Path：(0,3),(3,2),(2,1),(1,0)
value：47.5

illegal输出：
Path：(0,1),(3,2),(2,1)
value：20
